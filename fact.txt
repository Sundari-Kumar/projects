Copy code

cd factorial_app
2. Install RSpec for Testing
Add rspec-rails to your Gemfile under the :development, :test group:

ruby
Copy code
group :development, :test do
  gem 'rspec-rails', '~> 7.0.0'
end
Run:

bash
Copy code
bundle install
rails generate rspec:install
3. Generate the Factorial Model
Run the following command to generate the model:

bash
Copy code
rails generate model Factorial number:integer result:integer
Run the migration:

bash
Copy code
rails db:migrate
4. Write the Test Case First (Failing Test)
Open spec/models/factorial_spec.rb and write a test to validate the factorial logic:

ruby
Copy code
require 'rails_helper'

RSpec.describe Factorial, type: :model do
  describe '#calculate' do
    it 'returns the correct factorial for a given number' do
      factorial = Factorial.new(number: 5)
      expect(factorial.calculate).to eq(120)
    end

    it 'returns 1 for 0 as input' do
      factorial = Factorial.new(number: 0)
      expect(factorial.calculate).to eq(1)
    end

    it 'returns 1 for 1 as input' do
      factorial = Factorial.new(number: 1)
      expect(factorial.calculate).to eq(1)
    end
  end
end
5. Implement the Code
Modify the Factorial model (app/models/factorial.rb) to pass the test:

ruby
Copy code
class Factorial < ApplicationRecord
  def calculate
    return 1 if number.zero?
    (1..number).inject(:*)
  end
end
6. Run the Tests
Run RSpec to verify the tests:

bash
Copy code
bundle exec rspec
You should see all tests passing.
